herkforth

The herkforth mission is to make computer use dramatically more productive and peaceful.

The herkforth goals lay out a strategy for furthering this mission by creating an operating system for PPC with many ideas from The Humane Interface, forth and colorforth.

CURRENT STATUS: Compiler, assembler, code editor and interpreter working under GNU/Linux (PPC only.) Initial support for booting (can't save yet though.) Active development by Jason Woofenden

tarball: http://herkamire.com/downloads/herkforth-0.7.tar.bz2 (see herkforth changes)

Binary (with scripts to run on x86 GNU/Linux with qemu): http://herkamire.com/downloads/herkforth-bin-0.7.tar.bz2

repository: http://jason.herkamire.com:5000/svn/herkforth/ WARNING: Not Always On

documentation: herkforth docs
