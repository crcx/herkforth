herkforth variables

herkforth's editor supports variables. The & key switches to "insert variable" color (white.) Variables are displayed in white followed by their current value.

In case it isn't obvious, you can set the value of a variable like so:

1) enter a value and press ENTER

2) enter the variable name and press ENTER

3) type ! and press ENTER
