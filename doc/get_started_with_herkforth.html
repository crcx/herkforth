Getting started with herkforth

You'll need to know at least a little about forth (and preferably colorforth as well) for herkforth to make much sense. If you don't, then you should first read one of the introductions on the forth documentation page.

First build herkforth

then type: ./herkforth000

enter a word or number and type ENTER (BTW, space will insert the word in the current block and not execute it)

press SHIFT-Q to quit.

Now read how to use the herkforth interface

And finally you may want to know some herkforth words
